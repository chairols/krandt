Krandt ACCESOS es la solución de control de accesos dentro de la suite de aplicaciones que provee Krandt.

Diseñamos la mejor solución que se adapte a las necesidades del cliente.

Disponemos de una amplia variedad de equipos para captura de fichadas los cuales se encuentran integrados a nuestra solución.

Utilizamos equipos IP de última generación tanto de proximidad como biometría manteniendo además compatibilidad con todas las tecnologías anteriores:

• Proximidad
• Biometría
• Banda Magnética
• Código de Barras

La base de datos está abierta para el cliente. Krandt accesos se encuentra integrado con Krandt T&A para realizar el cálculo y liquidación de horas trabajadas.
